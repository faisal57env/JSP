Character Entity / HTML Entities 

        

            1. Character entities start with "&" and ends with a ";" 
            2. Character entities are used to display the Characters which are not present on the keyboard
            3. We can access the reserved the Special characters like < and > by using the character entities 
        

    
        this is an Example ↓
        

        < h1 > Hello World < /h1 >
